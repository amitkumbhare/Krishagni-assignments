Bash Script Example

This bash script prompts the user for a host name, user credentials, and the location of a file to transfer via SFTP. It then uploads the specified file to the specified location on the remote host using the SFTP protocol.

Script:

		#***************REMOVE THIS IS COMMENT LINE*********************#
		#!/bin/bash

		read -p "Enter the Host name: " HOST #example "192.168.122.132"
		echo "You entered $HOST"
		USER="ftpuser"
		PASSWORD=""

		DESTINATION=$1

		read -p "Enter your 1st file location: " ALL_FILES #example "DIR1/*"
		echo "You entered $ALL_FILES location"

		sftp -inv $HOST <<EOF
		user $USER $PASSWORD
		cd $DESTINATION
		put $ALL_FILES
		bye
		EOF
		#***************REMOVE THIS IS COMMENT LINE*********************#

The output of the script is displayed in the terminal.

Explanation:

  • The first line, #!/bin/bash, indicates that this is a bash script.
  • The read command prompts the user to enter the host name and stores the input in the variable $HOST.
  • The echo command outputs a message to the user, indicating the host name they entered.
  • The USER variable is set to "ftpuser" and the PASSWORD variable is left blank for security reasons, as it will be entered by the user when prompted.
  • The DESTINATION variable is set to the first command-line argument passed to the script.
  • The read command prompts the user to enter the location of the file to transfer, which is stored in the variable $ALL_FILES.
  • The sftp command is then used to initiate an SFTP session with the specified host.
  • The session commands are enclosed in a "here document" that starts with "<<EOF and ends with EOF. This is a way to provide input to the sftp command without having to type it on the command line.
  • The user command is used to authenticate the user with the specified host.
  • The cd command is used to change the current directory on the remote host to the specified destination directory.
  • The put command is used to transfer the specified file to the remote host.
  • The bye command is used to end the SFTP session.

Script Usage

To use this script, save it as a .sh file and make it executable using the chmod command. Then, run the script with the destination directory as the first command-line argument.

For example, if the script is saved as "upload_script.sh" and the destination directory is "/home/ftpuser/upload/", you would run the script as follows:

./upload_script.sh /home/ftpuser/upload/

The script will prompt you for the host name, the location of the file to transfer, and the password for the user account on the remote host.
