Bash Script Example

This script checks if the provided input is a file or directory and performs actions accordingly.

Script:

#!/bin/bash

for x in $*
do
	if [[ -f $x ]]
	then
		echo "$x is a file."
		echo "no of line in $x file are: "
		wc -l $x 
		NO=`wc -l $x`
		read lineno
		if [[ $NO > $lineno ]]
		then
			head -n $lineno	$x
		else
			echo "Range Excided."
		fi
	elif [[ -d $x ]]
	then
		echo "$x is a directory."
	else
		echo "enter vaild file name or directory."
	fi
done

The output of the script is displayed in the terminal.

Explanation:

  • The script uses a for loop to iterate over the command-line arguments provided.
  • It checks if the current argument is a file using the -f option with the [[ command.
  • If the current argument is a file, it prints a message with the file name and the number of lines in the file using the wc -l command.
  • It then asks the user to enter a line number and prints the first n lines of the file using the head command, where n is the line number entered by the user.
  • If the user enters a line number that is greater than the number of lines in the file, it prints a message saying "Range Exceeded".
  • If the current argument is a directory, it prints a message with the directory name.
  • If the current argument is neither a file nor a directory, it prints a message asking the user to enter a valid file name or directory.
